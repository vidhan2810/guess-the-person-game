# Guess The Person Game 🎮

A fun mini‑website where you guess the person based on 5 hints.  
You get 3 attempts to guess correctly!

## How to Play
- Click **Next** to reveal hints one by one.
- Enter your guess in the box.
- You have 3 attempts to get it right.
